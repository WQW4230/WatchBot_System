# STM32 WatchBot_Controi
一个基于 STM32F103C8T6 的多任务嵌入式系统，包含小游戏、多级菜单、三轴机械臂、rtc 实时时钟等红外遥控控制功能的项目。

## 功能
- 通讯录、扫雷、贪吃蛇、猜数字小游戏
- 三轴机械臂控制（ADC+DMA+双摇杆+红外遥控控制） 支持平滑移动和快速移动
- 可调速直流电机风扇，支持正反转
- RTC 实时时钟
- 定时闹钟提醒
- 红外遥控菜单选择
- 可调速LED
- 可调蜂鸣时间与开关控制
- OLED 显示 信息、菜单及状态显示
